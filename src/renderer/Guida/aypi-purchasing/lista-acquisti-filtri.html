Ricerca e filtri

  1. Nella barra Ricerca scrivi il nome del prodotto, un tag o un dipendente.
  2. Nel menu Priorità scegli: Tutte, Alta, Media o Bassa.
  3. Nel menu Tag seleziona una categoria specifica.
  4. Nel menu Ordina per scegli data (nuove/vecchie) o nome.
Consiglio

Combina ricerca + tag per trovare rapidamente tutte le righe simili.
