Durata e archiviazione interventi

  • Una riga convalidata resta visibile in lista per circa 30 giorni.
  • Una riga eliminata resta visibile in lista per circa 7 giorni.
  • Dopo queste soglie la riga viene rimossa dalla vista operativa.

Pulizia debug (admin)

Il pulsante debug rimuove subito dal JSON operativo tutte le righe già convalidate o eliminate.

Nota

La retention riguarda la visualizzazione e il file operativo corrente; usa export/backup per conservazione storica.
