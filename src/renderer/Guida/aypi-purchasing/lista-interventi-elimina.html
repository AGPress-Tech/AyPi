Eliminazione riga intervento

Gli admin possono eliminare tutte le righe. I dipendenti possono eliminare solo le proprie richieste finché un admin non le ha già gestite (modifica o convalida/elimina).

  1. Clicca l'icona X sulla riga.
  2. Conferma nel popup.
  3. La riga viene marcata come eliminata.

Nel popup di conferma puoi usare anche Invio come OK.

  • Se un admin elimina una richiesta di un dipendente, è richiesta una motivazione.
  • Passando il mouse sulla riga eliminata compare: Admin: X Y e Motivazione.
  • Se è l'utente ad annullare, il tooltip mostra Eliminato dall'utente.
Attenzione

Le righe eliminate restano visibili solo temporaneamente e poi vengono rimosse dal JSON operativo.
