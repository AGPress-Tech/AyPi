# AyPi Development

## Prerequisites
- Node.js (LTS)
- npm

## Install
```bash
npm install
```

## Run
```bash
npm start
```

## Scripts
- `npm start`: run the app in dev mode
- `npm run build`: build with electron-builder

## Common paths
- Main entry: `main.js`
- Renderer scripts: `scripts/`
- HTML views: `pages/`
