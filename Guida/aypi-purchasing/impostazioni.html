AyPi Purchasing / Impostazioni

Impostazioni

  • Tema: light, dark o AyPi.
  • Gestione dipendenti: stessi dati del calendario AyPi.
  • Gestione admin: aggiunta/modifica/rimozione admin con password.
  • Gestione categorie: categorie e colori dei tag (solo admin).
  • Tipologie interventi: elenco tipologie per richieste e lista interventi (solo admin).

Dettagli utili

  • I dati dipendenti e admin sono condivisi con il calendario.
  • Le modifiche admin richiedono conferma password.
  • Nei popup password puoi usare Invio per confermare.
