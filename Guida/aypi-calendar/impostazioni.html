Impostazioni

Dal pulsante a forma di ingranaggio (in alto a destra) accedi alle opzioni tematiche e alle funzioni per admin.

Procedura

  1. Apri Impostazioni (icona ingranaggio)
  2. In Tema seleziona: chiaro, scuro o tema AyPi
  3. In Mostra Festività Future vedi l’elenco delle festività inserite
  4. In Mostra Chiusure Future vedi l’elenco delle chiusure inserite
  5. Usa Configurazione per scegliere quali operazioni richiedono il login admin (richiede sempre password)
  6. Usa Gestione Admin solo per operazioni particolari (richiede login admin e password)
  7. Usa Backup per creare o ripristinare una copia del calendario (richiede login admin)
Solo Admin

Configurazione, Gestione Admin e Backup richiedono sempre la password admin.

Note
  • Le sezioni di gestione admin sono pensate per casi specifici e operazioni sensibili.
