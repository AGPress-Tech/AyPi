Gestione Admin, Panel Control e Mailing

Nelle impostazioni puoi accedere all’area di configurazione amministratori (richiede login admin e password ogni volta).

Procedura

  1. Apri Impostazioni ? Gestione admin
  2. L’amministratore principale può creare admin secondari con Aggiungi Admin
  3. In modifica puoi aggiornare: nome, cognome, email, password e (opzionale) telefono
  4. In rimozione puoi eliminare un admin, ma deve rimanerne almeno uno attivo
  5. Usa recupero password per inviare un codice a 6 cifre (valido 5 minuti) via email
  6. Inserisci il codice e imposta la nuova password
Solo Admin

Questa funzione e disponibile solo dopo il login admin e richiede password ogni volta.

Note
  • L’app impedisce la cancellazione dell’ultimo amministratore.
  • L’azzeramento totale degli admin è possibile solo tramite riconfigurazione in modalità sviluppatore (non documentata).
