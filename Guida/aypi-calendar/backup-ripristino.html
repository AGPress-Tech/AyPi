Backup e Ripristino

La funzione Backup si trova nelle Impostazioni e richiede il login admin. Consente di creare una copia completa dei file calendario e di ripristinare un backup precedente.

Backup calendario

  1. Apri Impostazioni (icona ingranaggio).
  2. Premi Backup (richiede login admin).
  3. Seleziona Backup calendario.
  4. Viene creata una cartella nel Server con nome AAAA-MM-GG (con suffisso se esiste già).

Recupera (ripristino)

  1. Apri Impostazioni e poi Backup.
  2. Premi Recupera.
  3. Seleziona la cartella del backup da ripristinare.
  4. Conferma: i file vengono copiati nella cartella originale e sovrascrivono quelli presenti.
Solo Admin

Questa funzione e disponibile solo dopo il login admin.

Note
  • Se l’ultimo backup ha più di 7 giorni, il sistema ne crea uno automatico.
  • Quando i backup superano 10, viene eliminato automaticamente quello più vecchio.
  • Il ripristino sovrascrive i file correnti: usalo con attenzione.
